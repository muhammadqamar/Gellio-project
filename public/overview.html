Galileo Program Master API
Galileo’s Program Master API is used by our Partners to create sophisticated card-based financial products. Almost any card product imaginable can be created and maintained with our API. The API leverages Galileo’s advanced platform making it easy and rewarding for developers to work with.
Tip The Sandbox mimics the Program Master’s functionality, in a no-risk environment. Try it before you buy it.
Tip Quick start guides walk developers through the basic use of the API. Reading through the guides will prepare any developer to use the API.
