Account
Use the Program Master API to create and maintain accounts.
Creating an account and issuing a card
In this guide, we will use the createAccount() endpoint to create a new account for instant issue cards. We will also load funds to the card with createPayment().
Prerequisites
  • An active Sandbox account.
  • A working development environment.
  • type parameter defined (with Galileo Processing).
A recap on permissions; the following must be set-up with Galileo before proceeding:
  • apiLogin - Provider's web service username.
  • apiTransKey - Provider's web service password.
  • providerId - Galileo issued provider identifier.
  • prodId - The identity of the account or card account being created.
  • transactionId - A string of alphanumeric characters (hyphen also allowed) that helps uniquely define your transaction on the Galileo Platform.
Recommendation : create a Python template (or your preferred language) of the permissions.
Steps
Goal for this guide : Successfully create a new account using the createAccount() method. Load funds onto the new account by using createPayment().
End product : A card with a $4,000.00 limit, created for "JJ Tecumseh", and loaded with $4,000.
Step one: Use createAccount() to create an account
Galileo's API is largely account based, so the first step is to create an account with a call to createAccount(). The following code shows you how to call createAccount(). We will get the PRN from the pmt_ref_no field.
The payment reference number (PRN) is used within Galileo's system to uniquely identify an account. This term may be used interchangeably with 'account number'. The provider is encouraged to store this number so it may be used in consecutive API calls.
In addition to the credentials; use the following parameters:
  • prodId=5094
  • id=000000000
  • idType=2
  • firstName=JJ
  • middleName=Ace
  • lastName=Tecumseh
  • loadAmount=4000
Use the code below to call createAccount() to create an account. Sample code for several languages are included.

// The following Java code will make a createAccount call
        // and print the XML response.
        
        import java.io.BufferedReader;
        import java.io.IOException;
        import java.io.InputStreamReader;
        import java.io.OutputStream;
        import java.io.Reader;
        import java.net.HttpURLConnection;
        import java.net.MalformedURLException;
        import java.net.URL;
        import java.net.URLEncoder;
        import java.util. *;
        
        class GalileoAPICall
        {
            public static void main(String[] args) {
                try {
                    Map<String,Object> params = new LinkedHashMap<>();
                   params.put("apiLogin", " ");
                   params.put("apiTransKey", " ");
                   params.put("providerId", "550");
                   params.put("transactionId", "40985-random-string-10473");
                   params.put("prodId", "501");
                    StringBuilder postData = new StringBuilder();
                    for (Map.Entry<String,Object> param : params.entrySet()) {
                        if (postData.length() != 0) postData.append('&');
                        postData.append(URLEncoder.encode(param.getKey(), "UTF-8"));
                        postData.append('=');
                        postData.append(URLEncoder.encode(String.valueOf(param.getValue()), "UTF-8"));
                    }
                    byte[] postDataBytes = postData.toString().getBytes("UTF-8");
        
                    URL url = new URL("https://sandbox-api.gpsrv.com/intserv/4.0/createAccount");
        
                    HttpURLConnection conn = (HttpURLConnection) url.openConnection();
                    conn.setRequestMethod("POST");
                    conn.setRequestProperty("Content-Type", "application/x-www-form-urlencoded");
                    conn.setRequestProperty("Content-Length", String.valueOf(postDataBytes.length));
                    conn.setDoOutput(true);
                    conn.getOutputStream().write(postDataBytes);
        
                    String xmlOutput;
        
                    Reader in = new BufferedReader(new InputStreamReader(conn.getInputStream(), "UTF-8"));
                    for (int c; (c = in.read()) >= 0;) {
                        System.out.print((char)c);
                    }
                } catch (MalformedURLException e) {
                    e.printStackTrace();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }

//   Creating a virtual card

The createVirtualCard() method is very similar to the createAccount() method. The createVirtualCard() differs from createAccount() in that it will only create an account/card combination with a virtual card product ID and the method returns full PAN, expiry, and CVV2. Similar to createAccount(), it also allows the loading of funds to the card at the time of creation. If a call to createVirtualCard() is made using a product ID that is not defined as a virtual card, an invalid product response will be returned.

Prerequisites

  • An active Sandbox account.
  • A working Python environment.

A recap on permissions; the following must be set-up with Galileo before proceeding:

  • apiLogin - Provider's web service username.
  • apiTransKey - Provider's web service password.
  • providerId - Galileo issued provider identifier.
  • prodId - The identity of the account or card account being created.
  • transactionId - A string of alphanumeric characters (hyphen also allowed) that helps uniquely define your transaction on the Galileo Platform.
